# coverage-router-ksa — public showcase

Public mirror of the HTML pages from [SaifDalli2/coverage-router-ksa](https://github.com/SaifDalli2/coverage-router-ksa) (private source repo). This repo exists so the showcase can be served via GitHub Pages without making the source repo public.

## Live URL

After GitHub Pages is enabled (Settings → Pages → Source: `main` branch, root):

**https://saifdalli2.github.io/coverage-router-ksa-public/**

## Pages

| Page | Description |
|---|---|
| `index.html` | Landing — redirects to `Research.html` |
| `Research.html` | Research showcase (English): rule families, coverage map, decision-flow diagram, regulator letter drafts, etc. |
| `Research.ar.html` | Arabic version of the research showcase (المسودة العربية) |
| `tester.html` | Interactive COB decision-engine tester (English) — load any of 25 worked scenarios, run the algorithm in your browser |
| `tester.ar.html` | Arabic version of the interactive tester |

## Source

The actual research, source PDFs, decision-tree definitions, regulator letter drafts, and the COB algorithm spec live in the **private** repo: [SaifDalli2/coverage-router-ksa](https://github.com/SaifDalli2/coverage-router-ksa).

This public repo only contains the rendered HTML showcase pages.

## Sync

Pages here are mirrored from `web/` in the source repo. When the source updates, the mirror needs to be re-pushed.
